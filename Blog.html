Important FAQ'S

What is Document Object Model (DOM)?

DOM is a programming interface that represents an HTML document as a tree structure.
It provides a way to access and manipulate elements, attributes, and text content of the document.

How do you select an element from the DOM?


getElementById(): Selects an element by its unique ID attribute.
querySelector(): Selects the first element that matches a CSS selector.
querySelectorAll(): Selects all elements that match a CSS selector.

What is event delegation in the context of the DOM, and why is it useful?

Event delegation is a technique in the DOM where you attach an event listener to a parent element instead of directly to each child element.
This improves performance, handles dynamic content efficiently, and simplifies code.

It's useful for many reasons like:
1. Improved performance
2. Dynamic content
3. Simplified code

How do you manipulate an element's attributes and styles using the DOM?

To manipulate attributes, use setAttribute() and getAttribute().

To manipulate styles, use style.propertyName or classList for class-based styling.
